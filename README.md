## **🌍 Project Overview / Proje Özeti**

**🇬🇧 [English]**
This project is a sophisticated exploration of **Computer Graphics**, blending mathematical precision with artistic aesthetics. Inspired by the Japanese **"Sumi-e"** style, it visualizes a dynamic scene using purely mathematical models instead of static assets.

**🇹🇷 [Türkçe]**
Bu proje, matematiksel hassasiyeti sanatsal estetikle birleştiren kapsamlı bir **Bilgisayar Grafiği** çalışmasıdır. Japon **"Sumi-e"** sanatından esinlenilerek hazırlanan sahnede, hazır görseller yerine tamamen matematiksel formüller kullanılmıştır.

---

## **🛠️ Technical Implementation / Teknik Uygulama**

* **Cubic Bézier Curves (Kübik Bézier Eğrileri):** * **EN:** All organic forms (the bird and petals) are rendered by manually calculating points using parametric equations, satisfying the core principles of vector graphics.
    * **TR:** Sahnedeki tüm organik formlar (kuş ve yapraklar), parametrik denklemler kullanılarak manuel hesaplanan noktalarla çizilmiştir.

* **Stochastic Systems (Stokastik Sistemler):** * **EN:** Implemented a generative system for the Sakura leaves, where positions, colors, and transparency are determined through random probability (stochastic modeling).
    * **TR:** Sakura yaprakları için konum, renk ve saydamlık değerlerinin olasılıksal olarak belirlendiği generatif bir sistem kurgulanmıştır.

* **Camera State Machine (Kamera Durum Makinesi):** * **EN:** Developed a dynamic zoom and follow system using a **State Machine** (Search, Zoom-In, Follow, Zoom-Out) to manage complex scene transitions.
    * **TR:** Sahne geçişlerini ve kuş takibini yönetmek amacıyla bir **Durum Makinesi** üzerinden dinamik kamera kontrol sistemi geliştirilmiştir.

## **🧠 Connection to AI & Computer Vision / Yapay Zeka ve Görüntü İşleme Bağlantısı**

### **🇬🇧 [English]**
* **Vectorization:** Understanding how mathematical curves represent data shapes—a fundamental concept in pattern recognition.
* **Pixel Manipulation:** Direct access to raw pixel data (RGBA) for digital reconstruction, mirroring image processing pipelines.
* **Stochastic Modeling:** Handling unpredictable behaviors through probability, a core principle of ML.
* **State Control:** Managing autonomous entity behavior using a **State Machine**—the logic behind autonomous agents.

### **🇹🇷 [Türkçe]**
* **Vektörizasyon:** Matematiksel eğrilerin veri formlarını nasıl temsil ettiğini kavrama; bu, örüntü tanıma süreçlerinin temel taşıdır.
* **Piksel Manipülasyonu:** Dijital yeniden yapılandırma için ham piksel verisine (RGBA) doğrudan erişim; bu, görüntü işleme süreçlerinin ilk adımını simüle eder.
* **Stokastik Modelleme:** Yapay zekanın temel prensibi olan belirsiz davranışların olasılık üzerinden modellenmesi.
* **Durum Kontrolü:** Otonom araçların arkasındaki mantık olan **Durum Makinesi (State Machine)** ile dijital bir varlığın davranışlarını yönetme.
